# BreastCancerMLAnalysis
Machine Learning Analysis of the [Breast Cancer Wisconsin Dataset](https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data).

---
To navigate the notebooks in an orderly matter take a look at:
- data-visualization.ipynb
- data-preprocessing.ipynb
- training.ipynb
- inference.ipynb

---
When loading the repository you can use the `requirements.txt` to load the required python packages with pip
